Trak TRL1540V Quick Start GuideÂ¶

The Trak TRL 1540V is a 2-axis computer numerical control (CNC) lathe. This guide is designed to get an individual up and running on the machine in the minimum amount of time and with little training. For complete documentation see the machine specific guide provided by Southwestern Industries. The following instructions are intended to be followed in order.
